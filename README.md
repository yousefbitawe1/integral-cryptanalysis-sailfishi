# Integral Cryptanalysis Defense for SAILFISH-I

This project analyzes the vulnerability of the SAILFISH-I lightweight block cipher against Integral Cryptanalysis.

A lightweight defense mechanism using Extra Key Whitening is proposed to disrupt the balanced property exploited by the attack.

## Project Contents
- report.docx (Project Report)
- presentation.pptx (Presentation Slides)
- simulation.py (Python Simulation)

## Description
The project uses a reduced-round simulation to demonstrate how structured plaintext sets behave under integral cryptanalysis. The results show that adding extra key whitening reduces the effectiveness of the attack by disrupting the balanced property.
